## Color Palette Reference

All animated backgrounds use the neon/vaporwave palette:

- **Dark Purple**: #0D0526 (base background)
- **Purple**: #B967FF
- **Pink**: #FF71CE (neon pink)
- **Cyan**: #01CDFE (electric cyan)
- **Yellow**: #FDFF6A (used in game objects, not backgrounds)

Colors are intentionally darkened in backgrounds (typically 20-30% brightness) to:
1. Not distract from gameplay elements
2. Maintain readability of text
3. Create depth (backgrounds recede, game objects pop)

## Performance Notes

All backgrounds render at **60 FPS** because:
- GPU handles all computation
- Fragment shaders are highly optimized
- No texture lookups (procedural generation)
- Minimal uniform updates (just time and resolution)

CPU overhead per frame:
- Setting uniforms: < 0.1ms
- Vertex processing: < 0.1ms
- Total: Negligible

## Switching Backgrounds

It's super easy! Just edit one line in `src/utils/constants.py`:

```python
# Choose your favorite:
BACKGROUND_TYPE = "starfield"  # Recommended default
# BACKGROUND_TYPE = "plasma"   # Psychedelic flow
# BACKGROUND_TYPE = "waves"    # Retro grid
# BACKGROUND_TYPE = "solid"    # Minimal distraction
```

No code compilation needed - just save and run!

## Which Background Should I Choose?

**Choose Starfield if:**
- You want the classic space aesthetic
- You like parallax scrolling effects
- You want something universally appealing

**Choose Plasma if:**
- You want a hypnotic, organic feel
- You like psychedelic visuals
- You want smooth, flowing motion

**Choose Waves if:**
- You're a synthwave/outrun fan
- You like retro 80s aesthetics
- You want rhythmic, regular patterns

**Choose Solid if:**
- You want absolute minimal distraction
- You need maximum performance
- You prefer classic simplicity

All backgrounds were carefully designed to be:
